1\. Identify the data type or class for each of the following values:

| **Value**          |
| ------------------ |
| `'True'`           |
| `False`            |
| `(1, 2, 3)`        |
| `1.5`              |
| `[1, 2, 3]`        |
| `2`                |
| `range(5)`         |
| `{1, 2, 3}`        |
| `None`             |
| `{'foo': 'bar'}`   |

**> Solution**

| **Value**          | **Class**          |
| ------------------ | ------------------ | 
| '`True'`           | `str`              |
| `False`            | `bool`             |
| `(1, 2, 3)`        | `tuple`            |
| `1.5`              | `float`            |
| `[1, 2, 3]`        | `list`             |
| `2`                | `int`              | 
| `range(5)`         | `range`            |
| `{1, 2, 3}`        | `set`              |
| `None`             | `NoneType`         | 
| `{'foo': 'bar'}`   | `dict`             |
