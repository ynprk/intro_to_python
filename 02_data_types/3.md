3\. Create a dictionary named `pets` that contains a list of pet names and the type of animal. For instance:


| **Name**       | **Animal**     |
| -------------- | -------------- |
| Asta           | dog            |
| Butterscotch   | cat            | 
| Pudding        | cat            |
| Neptune        | fish           |
| Darwin         | lizard         |

You can use that data or make up your own.

**> Solution**
```
pets = {
    'Asta': 'dog', 
    'Butterscotch': 'cat',
    'Pudding': 'cat', 
    'Neptune': 'fish', 
    'Darwin' 'lizard',
}
```
